## Shortest Subarray with Sum at Least K

This problem asks you to
find the length of the shortest non-empty
contiguous subarray within an integer array nums 
whose elements sum to at least a given target value
k. If no such subarray exists, you should return -1.

The complete solution explanation to my post is [here](https://francescogalazzo.github.io/code-blog/posts/min-subarray-sum-with-negatives/).
